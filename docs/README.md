# Vitaledge サイト管理ガイド

このフォルダは、サイトの管理・運用に必要な情報をまとめたものです。

## ドキュメント一覧

| ファイル | 内容 |
|---|---|
| [01_setup.md](01_setup.md) | 初期セットアップ（初回のみ） |
| [02_articles.md](02_articles.md) | 導入事例の追加・編集 |
| [03_deploy.md](03_deploy.md) | ビルドとデプロイ |

## このサイトについて

- **フレームワーク**: Next.js（App Router）
- **スタイリング**: Tailwind CSS
- **コンテンツ管理**: `content/cases/` 以下の Markdown ファイル
- **記事の追加**: コードを書かずに `.md` ファイルを置くだけで反映される

## 作業の全体フロー

```
記事を追加・編集したい
  └→ 02_articles.md を読む

サイトを本番に反映したい
  └→ 03_deploy.md を読む

初めてこのプロジェクトを触る
  └→ 01_setup.md から読む
```
